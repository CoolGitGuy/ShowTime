# ShowtimeStarter

Minimal Kotlin Multiplatform Compose starter for the `Showtime` project.

Included:
- Compose Multiplatform setup
- Koin
- Ktor + Ktorfit
- Coil 3
- Navigation
- Napier

Intentionally not included yet:
- Room
- DataStore
- Auth flow
- MVI implementation
- Repository/DAO code

This is meant to be a clean starting point so you can rebuild the architecture from scratch.
